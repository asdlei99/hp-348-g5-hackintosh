# HP 348 G5 黑苹果配置

### 2020.07.31 更新:

- 支持 MacOS Big Sur beta3
- OpenCore 升级为最新编译的 0.6.0 版本
- 更新内置驱动程序版本
- 新增 `AppleAHCIPort.kext` 以解决 Big Sur 不识别内置硬盘问题
- 移除 i2c 相关驱动（触摸板暂时不可使用）
